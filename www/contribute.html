Contribute to LLM Observatory

LLM Observatory is an open-source, community-driven initiative dedicated to monitoring, mapping, and understanding large language models. We welcome contributors from all backgrounds—whether you're a researcher, engineer, designer, or simply curious about AI.

Contributions can range from improving benchmark coverage and UI design, to writing documentation, analyzing model behavior, or proposing new evaluation tasks. We aim to be transparent, collaborative, and rigorous.

If you're interested in getting involved, you can:

  • Submit an issue or pull request on our GitHub
  • Join discussions or propose ideas for new features
  • Help us document evaluations and expand coverage
